# Web Breakpoint Recipes

## Desktop

- Preserve richer adjacency such as sidebars, secondary panels, and multi-column card grids.
- Use width constraints to keep text readable.

## Tablet

- Reduce columns before shrinking content too aggressively.
- Re-evaluate toolbar density and filter layouts.
- Allow more sections to stack vertically if comparison is not the main goal.

## Mobile

- Stack major regions vertically.
- Convert side-by-side actions into wrapped or stacked action groups.
- Keep primary actions prominent and reachable.
- Use full-width controls when they improve clarity and touch comfort.

## Common reflow patterns

- Two-column content to stacked sections
- Four-card grid to two-card grid to one-card stack
- Inline control bars to wrapped chips or segmented rows
- Sidebar filters to drawer or collapsible panel
