Place licensed Artgrid MP4 files in this folder using the naming convention below.

Each frame uses 5 clips and rotates to the next clip every 5 seconds, then loops
back to clip 1 after clip 5.

Required files:

- `vista-1.mp4` to `vista-5.mp4`
- `portrait-1.mp4` to `portrait-5.mp4`
- `studio-1.mp4` to `studio-5.mp4`
- `clinic-1.mp4` to `clinic-5.mp4`
- `emerald-1.mp4` to `emerald-5.mp4`
- `rose-1.mp4` to `rose-5.mp4`
- `sand-1.mp4` to `sand-5.mp4`
- `noir-1.mp4` to `noir-5.mp4`

These sequences map to the existing frame variants used across the hero and
service grid.
